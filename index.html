01 — HARDWARE LAYER

IoT Components

Industrial-grade hardware stack powering real-time location intelligence at the edge.

📡

GPS Module

Captures precise latitude/longitude coordinates using satellite triangulation. Works globally with no subscription fees.

NEO-6M / NEO-8M GPS · NMEA 0183 Protocol · ±1-3m Accuracy
📶

GSM / Wi-Fi Module

Transmits location data to the backend via cellular networks (SIM800L) or local Wi-Fi (ESP8266 onboard) for flexible connectivity.

SIM800L GSM · ESP8266 Wi-Fi · MQTT / HTTP Protocol
🧠

Microcontroller

The brain of the device — processes GPS data, manages power, and orchestrates data transmission through programmable logic.

Arduino Uno / NodeMCU ESP8266 · 80MHz · 4MB Flash
🔋

Battery & Power Supply

Rechargeable LiPo battery with TP4056 charging controller and 3.3V/5V step-down regulator for stable, long-duration operation.

3.7V 2500mAh LiPo · TP4056 Charger · 48h+ Runtime
⚡

Hardware Data Flow

GPS acquires satellite fix → MCU parses NMEA strings → GSM/Wi-Fi encodes and transmits JSON payload → Backend API stores coordinates → Frontend renders live on map.

GPS_FIX → NMEA_PARSE
→ JSON_ENCODE
→ HTTP_POST /api/location
→ DB_WRITE coordinates
→ WEBSOCKET_PUSH client
→ MAP_UPDATE marker

05 — SECURITY

Enterprise-Grade Protection

Multi-layer security architecture protecting every request from device to database.

🔑

User Authentication

JWT tokens with short expiry, refresh token rotation, and device fingerprinting to prevent session hijacking.

JWT · HS256 · 15min Access Token
🔒

Password Encryption

bcrypt hashing with work factor 12, salt generation per user. Plaintext passwords never stored or logged.

bcrypt · Salt Rounds: 12 · PBKDF2
🗄️

Secure Database

Parameterized queries prevent SQL injection. Role-based access control isolates user data completely.

Prepared Statements · RBAC · Audit Logs
🌐

HTTPS / TLS 1.3

All client-server communication encrypted. HSTS headers enforced. Certificate pinning on mobile clients.

TLS 1.3 · HSTS · Perfect Forward Secrecy
🛡️

AES-256 Data Encryption

GPS coordinates and sensitive device metadata encrypted at rest using AES-256-GCM before database storage.

AES-256-GCM · Encrypted at Rest
⚡

Rate Limiting & DDoS

API rate limiting per IP and per user token. Nginx-level request throttling with automatic IP banning for abuse.

100 req/min · Fail2Ban · Nginx WAF
